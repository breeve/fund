# 地理层级划分

行政区->板块->生活圈->小区

| 层级 | 名称   | 示例                           | 说明                   |
| ---- | ------ | ------------------------------ | ---------------------- |
| 1    | 行政区 | 南山区、宝安区                 | 预算筛选               |
| 2    | 板块   | 科技园、碧海、红山             | 通勤+配套核心决策      |
| 3    | 生活圈 | 深大花园周边、碧海湾地铁站周边 | 步行可达范围，共享配套 |
| 4    | 小区   | 具体楼盘                       | 最终决策               |
