# 资产分类

- 金融理财
  - 基金: 公募基金, 类型: A股, H股, AH, 中美, 全球
  - 私募: 固定大额理财, 类型: 债券策略-长债, 债券策略-短债, 多策略-激进, 多策略-保守, 股票策略
  - 策略: 全球精选90, 全球精选100, 股票基金金选, 行业景气度策略, 百分百进攻
  - 定期: 银行定期, 股票定期
  - 活钱: 余额宝, 活期存款
  - 金融衍生品: 黄金
- 保障
  - 住房公积金
  - 个人公积金

标签, 金融理财需要提供标签功能

- 地域划分: 国内, 全球, 港股
- 风险等级: xx, xx

时间, 所有资产, 都需要一个录入时间,这样后续好评估资产变化趋势
