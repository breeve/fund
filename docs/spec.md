# 家庭资产管理系统 — 产品规格文档

## 一、功能概述

| # | 功能模块 | 核心价值 |
|---|----------|----------|
| 1 | 资产录入 | 建立完整的家庭资产负债表 |
| 2 | 资产展示 | 可视化呈现资产分布与变化趋势 |
| 3 | 资产分析 | 多维度量化评估财务健康状况 |
| 4 | 基金诊断 | 穿透分析基金持仓，评估组合适配度 |

## 二、资产分类体系

### 2.1 分类原则

采用三维分类模型，从三个独立维度对资产进行标记，避免单维度分类导致的归属歧义：

- **流动性维度**：资产变现的难易程度与时间周期
- **形态维度**：资产的存在形式
- **功能维度**：资产在家庭财务中的主要作用

> 注：一个资产可同时属于多个维度的子类别。例如：公积金账户（住房公积金）= 长期锁定 + 货币形态 + 增值/保障双重功能。

### 2.2 五大类别

---

#### 类别一：流动资产 (Liquid Assets)

> **流动性维度**: 高度流动（变现周期 < 7 天）
> **核心特征**: 流动性强、风险极低、随时可用

| 子类 | 包含内容 | 典型场景 |
|------|----------|----------|
| 现金类 | 手头现金、活期存款 | 日常开支、紧急备用 |
| 短期理财 | 7 天内可赎的理财产品、货币基金、国债逆回购 | 闲置资金临时打理 |
| 可快速变现类 | T+1 到账的定期存款、短期国债、通知存款 | 短期大额支出预备 |

**数据字段**：

- 名称、金额、开户机构、产品名称
- 年化收益率（仅供参考，非保证）
- 最近一次变动日期
- 备注 / 标签

**用途**: 作为家庭的「应急备用金」，覆盖 3–6 个月生活支出，应对突发的医疗、失业、意外等不确定性支出，确保财务「底气」。

**录入建议**:

- 金额精度：精确到分
- 收益率：填写当前持有期年化收益率
- 变动频率：每笔资产需记录最近一次变动日期（购买/存入/赎回）

---

#### 类别二：固定资产 (Fixed Assets)

> **流动性维度**: 低流动（变现周期 > 3 个月）或不可流通
> **核心特征**: 价值较高、长期持有、具备抗通胀属性

| 子类 | 包含内容 | 收益类型 |
|------|----------|----------|
| 自用型实物 | 自住房产、自用车辆、艺术品收藏、古董 | 消费属性（自用价值） |
| 投资型实物 | 用于出租的房产（住宅/商铺）、版权/专利、商标权 | 租金收入 / 许可费 |
| 投资型权益 | 有限责任公司股权（非上市）、合伙人权益 | 分红 / 转让收益 |

**数据字段**：

- 名称、购买日期、购买金额、当前估值 / 市价
- 持有目的（自用 / 投资）
- 当前状态（自住 / 出租 / 空置）
- 月租金（如适用）、租约到期日（如适用）
- 贷款信息（贷款金额、利率、剩余期限、月供，如适用）
- 折旧年限（车辆等）

**用途**: 作为财富的「压舱石」，提供长期价值稳定。投资型固定资产可通过租金、版税、分红等持续产生被动收入。

**录入建议**:

- 房产估值：每半年更新一次市场价（或由系统提示更新）
- 车辆按购入价格或折旧后净值录入
- 出租房产需同时记录对应贷款（如有），作为关联负债

---

#### 类别三：金融投资资产 (Financial Investment Assets)

> **流动性维度**: 中等流动（变现周期 1–30 天）
> **核心特征**: 追求资产增值、风险与收益并存

| 子类 | 包含内容 | 风险等级 |
|------|----------|----------|
| 权益类 | A 股、港股、美股、ETF（股票型）、REITs | 高 |
| 固定收益类 | 债券、国债、企业债、银行理财（固收类） | 中低 |
| 基金类 | 公募基金、私募基金、FOF、MOM、基金投顾 | 中–高 |
| 另类投资 | 黄金、实物贵金属、大宗商品、信托产品、私募股权 | 高 |
| 衍生品 | 期权、期货、结构性存款（保本/非保本） | 高 |

**数据字段**：

- 名称、产品代码（如适用）、持有数量 / 份额
- 成本单价、当前单价、成本总额、当前市值
- 买入日期、分红/分红再投资记录
- 基金：基金公司、基金类型（主动/指数）、管理费、托管费
- 股票：所属交易所、所属行业、权重（相对总市值占比）
- 账户所在机构

**用途**: 作为财富的「增长引擎」，通过大类资产配置分散风险，追求长期复利增长，跑赢通胀。

**录入建议**:

- 基金持仓需支持按「基金」维度查询，也需支持按底层持仓「行业/个股」维度查询（基金诊断功能）
- 股票/ETF 建议关联对应指数，便于后续与基准对比
- 持有期收益率与年化收益率应自动计算

---

#### 类别四：保障类资产 (Protection Assets)

> **流动性维度**: 长期锁定（理赔前不可变现，特殊情况可退保）
> **核心特征**: 风险转移、对冲极端损失

| 子类 | 包含内容 | 触发条件 |
|------|----------|----------|
| 健康险 | 重疾险、医疗险、防癌险 | 确诊约定疾病 / 发生医疗费用 |
| 意外险 | 意外伤害险、意外医疗险 | 意外事故发生 |
| 寿险 | 定期寿险、终身寿险、两全保险 | 身故 / 全残 |
| 年金险 | 养老年金险、教育金险、分红险 | 约定年龄 / 约定事件 |
| 社保 / 公积金 | 养老保险账户余额、医保账户余额、公积金账户余额 | — |

**数据字段**：

- 产品名称、保险公司、保单号
- 险种类别、保障额度 / 保额
- 年缴保费、缴费年限、已缴费年限、剩余缴费年限
- 保障期限（终身 / 至某年龄 / 一年期）
- 现金价值（如有）
- 等待期、责任免除（关键摘要）

**用途**: 作为家庭的「安全网」，防止突发重大疾病或意外掏空积蓄，保障基础生活不被改变。社保和公积金作为国家强制性保障，计入保障类资产能更完整地评估家庭防护水平。

**录入建议**:

- 年金险的「当前现金价值」计入净资产，但需注明流动性限制
- 社保养老账户余额可通过各地社保局查询入口获取（或手动录入）
- 公积金账户余额计入净资产，标注为「长期锁定」

---

#### 类别五：负债与权益调整 (Liabilities & Net Worth Adjustment)

> **流动性维度**: —
> **核心特征**: 影响资产净值的减项

| 子类 | 包含内容 | 性质判定 |
|------|----------|----------|
| 住房负债 | 住房贷款（商业贷款、公积金贷款、组合贷） | 良性（低息、资产增值潜力大） |
| 消费负债 | 车贷、消费贷、助学贷 | 视利率与用途判定 |
| 信用负债 | 信用卡欠款、消费分期、信用贷款 | 恶性（高息） |
| 其他负债 | 民间借贷、担保债务 | 视具体情况判定 |

**良性 vs 恶性负债判定规则**:

```
良性负债:
  - 贷款利率 < 5%（基准利率附近或以下）
  - 用于购买能增值或产生持续收益的资产
  - 月供占月收入比例 < 40%

恶性负债:
  - 信用卡分期利率 > 15%
  - 消费贷利率 > 市场平均水平
  - 用于纯消费、无资产增值可能
```

**数据字段**：

- 债务名称、债权人、贷款总额
- 已还本金、剩余本金、剩余期限
- 贷款利率（LPR 浮动或固定）、还款方式（等额本息/等额本金/先息后本）
- 月供金额、下次还款日
- 抵押物（如有）、是否已抵押
- 性质标签（良性 / 恶性 / 待定）

**用途**: 通过区分良性负债与恶性负债，优化整体债务结构，降低财务成本，将节省的利息转化为资产积累。

## 三、功能模块详细说明

### 3.1 资产录入

#### 录入流程

```
选择类别 → 填写基本信息 → 填写财务数据 → 补充关联数据 → 保存审核
```

#### 支持方式

- **手动录入**: 用户逐笔输入资产信息
- **批量导入**: 支持 Excel/CSV 模板批量导入
- **账户授权**: 经用户授权后，通过正规 API 拉取部分数据（如基金持仓、股票账户，需明确告知数据用途并脱敏处理）

#### 必填与选填

| 类别 | 必填字段 | 选填字段 |
|------|----------|----------|
| 流动资产 | 名称、金额 | 收益率、机构、备注 |
| 固定资产 | 名称、购买日期、估值 | 贷款信息、租金、折旧 |
| 金融投资 | 名称、成本、市值 | 行业、分红记录、跟踪指数 |
| 保障类 | 产品名称、保额、年缴保费 | 现金价值、理赔记录 |
| 负债 | 名称、总额、剩余本金、利率 | 抵押物、月供记录 |

#### 数据校验规则

- 金额：必须为正数，精确到分
- 日期：不得晚于当前日期
- 贷款利率：范围 0%–36%（超出范围提示确认）
- 月供：不得大于月收入（超过时预警）
- 保额：整数，万元为单位

### 3.2 资产展示

#### 核心视图

| 视图 | 内容 |
|------|------|
| 资产负债表 | 总资产、总负债、净资产，关键科目汇总 |
| 资产构成图 | 五大类别金额与占比（环形图/树状图） |
| 金融资产明细 | 股票/基金/债券分类明细列表 |
| 负债一览 | 全部负债明细，含良性/恶性标注 |
| 净资产趋势图 | 按月/季度/年度展示净资产变化曲线 |
| 资产流动性分布 | 高度流动 → 长期锁定的金字塔图 |

#### 交互要求

- 支持按时间区间筛选（近1月/近3月/近1年/全部）
- 支持按类别折叠/展开
- 净资产数据变更需有变化量标注（+XX万 / -XX万）
- 图表支持导出（PNG / CSV）

### 3.3 资产分析

#### 分析维度

| 分析维度 | 核心指标 | 诊断逻辑 |
|----------|----------|----------|
| **净资产分析** | 净资产总额、环比增长率 | 趋势判断、异常波动提醒 |
| **资产配置分析** | 五大类占比、风险资产占比、实物/金融资产比 | 配置是否失衡（参考值：风险资产 30%–70%，房产占比 < 60%） |
| **流动性分析** | 流动比率、应急储备充足率 | 流动资产能否覆盖 6 个月支出 |
| **偿债能力分析** | 资产负债率、月供收入比、恶性负债占比 | 偿债压力是否过重（预警线：月供 > 40% 月收入） |
| **保障充足度分析** | 人身险保障额 / 年收入、保障缺口 | 是否存在重大风险敞口 |
| **抗通胀能力分析** | 固定资产占比、权益类占比、黄金占比 | 资产实际购买力保值能力 |
| **分散度分析** | 资产相关性矩阵、跨类别集中度 | 鸡蛋是否放在同一个篮子 |
| **增长健康度分析** | 综合年化收益率 vs 通胀率 | 资产是否真正实现增值 |

#### 诊断报告输出

每项分析输出：

- **指标名称**与**当前值**
- **参考区间**（正常 / 警示 / 危险）
- **趋势图**（时间维度变化）
- **风险提示**（如有问题，给出原因简述）
- **优化建议**（2–3 条可操作的改进方向）

#### 预警机制

| 预警等级 | 触发条件 | 通知方式 |
|----------|----------|----------|
| 🟢 正常 | 指标在健康区间 | — |
| 🟡 关注 | 指标接近临界值 | 应用内通知 |
| 🔴 预警 | 指标超出健康区间 | 应用内通知 + 定期报告推送 |
| ⚠️ 危险 | 净资产骤降 > 20% 或流动性断裂 | 即时通知 |

### 3.4 基金诊断

#### 持仓穿透分析

- 解析基金定期报告（季报/年报）提取前十大重仓股/重仓债券
- 关联行业分类，计算基金行业分布
- 计算基金与用户现有持仓的**重叠度**（同一股票/行业占比）

#### 风险评级

基于以下维度综合打分（1–5 星）：

- 波动率（年化标准差）
- 最大回撤
- 夏普比率
- 与大盘相关性
- 基金经理稳定性

#### 适配度评估

针对用户整体资产组合，输入单只/多只基金，输出：

- 该基金与现有持仓的风险相关性
- 建议配置比例区间
- 费用合理性评估（管理费 + 托管费 vs 同类均值）

#### 组合模拟

- 录入目标基金后，模拟加入基金后的组合变化
- 输出新的资产配置比例、预期收益率、波动率

## 四、附录

### A. 术语表

| 术语 | 定义 |
|------|------|
| 净资产 | 总资产 − 总负债 |
| 流动资产 | 7 天内可变现且变现损失极小的资产 |
| 风险资产 | 价格波动可能导致本金损失的投资资产（股票、基金、黄金等） |
| 保障缺口 | 家庭需要但尚未通过保险覆盖的风险敞口 |
| 复利增长 | 收益再投资带来的指数级增长效应 |
| 夏普比率 | (组合收益率 − 无风险利率) / 组合波动率，衡量风险调整后收益 |
| 最大回撤 | 组合从最高点到最低点的最大跌幅 |

### B. 参考指标基准值

| 指标 | 建议范围 | 说明 |
|------|----------|------|
| 应急储备 | 3–6 个月支出 | 流动资产覆盖月数 |
| 资产负债率 | < 50% | 负债占资产比例 |
| 月供收入比 | < 40% | 月供占月收入比例 |
| 风险资产占比 | 100 − 年龄 ± 10% | 简单参考公式 |
| 房产占比 | < 60% | 固定资产中房产比例 |
| 保障额度 | 年收入 5–10 倍 | 人身险保额参考 |
