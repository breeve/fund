# 地理层级划分

行政区->板块->生活圈->小区

| 层级 | 名称   | 示例                           | 说明                   |
| ---- | ------ | ------------------------------ | ---------------------- |
| 1    | 行政区 | 南山区、宝安区                 | 预算筛选               |
| 2    | 板块   | 科技园、碧海、红山             | 通勤+配套核心决策      |
| 3    | 生活圈 | 深大花园周边、碧海湾地铁站周边 | 步行可达范围，共享配套 |
| 4    | 小区   | 具体楼盘                       | 最终决策               |

# 行政区

## 1. 基础信息

| 属性            | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| 行政区划代码    | 如 440305（南山区），对应国家标准 GB/T 2260 / 统计用区划代码 |

| 学区划分      | 哪个小区对哪个学校（最关键但最难获取） |
| 学位预警      | 教育主管部门发布的学位紧张程度预警     |

## 3. 交通配套

| 属性            | 说明                   |

| 道路拥堵指数    | 实时路况和历史拥堵数据 |

## 4. 生活配套

| 属性           | 说明                     |
| -------------- | ------------------------ |

| 商场、农贸市场 | POI 位置和评分           |

## 5. 房价行情

| 属性             | 说明                   |
| ---------------- | ---------------------- |
| 均价、成交量     | 月度统计，区域行情基准 |
