# 地理层级划分

行政区->板块->生活圈->小区

| 层级 | 名称   | 示例                           | 说明                   |
| ---- | ------ | ------------------------------ | ---------------------- |
| 1    | 行政区 | 南山区、宝安区                 | 预算筛选               |
| 2    | 板块   | 科技园、碧海、红山             | 通勤+配套核心决策      |
| 3    | 生活圈 | 深大花园周边、碧海湾地铁站周边 | 步行可达范围，共享配套 |
| 4    | 小区   | 具体楼盘                       | 最终决策               |

# 行政区

## 1. 基础信息

| 属性            | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| 行政区划代码    | 如 440305（南山区），对应国家标准 GB/T 2260 / 统计用区划代码 |
| 面积、人口、GDP | 区域基本面，用于板块横向比较                                 |
| 街道/社区边界   | 地理边界 polygon，支撑地图可视化和空间分析                   |

## 2. 教育资源

| 属性          | 说明                                   |
| ------------- | -------------------------------------- |
| 学校清单      | 幼儿园、小学、初中、高中、大学         |
| 学校评级/排名 | 省一级、市一级学校评估等级             |
| 学区划分      | 哪个小区对哪个学校（最关键但最难获取） |
| 学位预警      | 教育主管部门发布的学位紧张程度预警     |

## 3. 交通配套

| 属性            | 说明                   |
| --------------- | ---------------------- |
| 地铁线路/站点   | 线路走向、站点位置坐标 |
| 公交线路密度    | 公交覆盖和频次情况     |
| 道路拥堵指数    | 实时路况和历史拥堵数据 |
| 停车场位置/数量 | 静态停车场 POI         |

## 4. 生活配套

| 属性           | 说明                     |
| -------------- | ------------------------ |
| 医院（等级）   | 三甲、公立、私立医院分类 |
| 商场、农贸市场 | POI 位置和评分           |
| 公园、文化设施 | 休闲娱乐配套             |
| 商圈           | 已有商圈，规划中商圈     |

## 5. 房价行情

| 属性             | 说明                   |
| ---------------- | ---------------------- |
| 均价、成交量     | 月度统计，区域行情基准 |
| 在售套数、挂牌价 | 实时供需状态           |
| 历史价格走势     | 3-5 年价格曲线         |

## 6. 规划信息

| 属性         | 说明                              |
| ------------ | --------------------------------- |
| 国土空间规划 | 用途分区（居住/商业/工业/绿地等） |
| 在建重大工程 | 地铁、学校、医院等民生项目        |
| 城市更新计划 | 旧改项目列表和进度                |
